# 🚀 MangaGallery PROJECT CONSTITUTION

## 1. Project Overview (概要)
- **Project Name**: MangaGallery
- **Core Value**: Noteに投稿しているAI漫画作品を、独自の世界観（Obsidian & Cursorのキャラ性）で魅力的に紹介し、ポートフォリオとして機能させる。
- **Target Audience**: AI漫画に興味がある層、Noteの読者、クリエイター仲間。
- **Design Vibe**: **Pop & Character-Driven**
    - Obsidian (黒/紫/ゴシック) と Cursor (赤/琥珀/スポーティ) の対比を活かす。
    - 見ていて楽しい、動きのあるデザイン。

## 2. MVP Scope (最小機能範囲)
- **Hero Section**: 
    - ファーストビューで「MangaGallery」のロゴとキャラクターを表示。
    - "Welcome to our world" 的なキャッチコピー。
- **Character Intro**:
    - **Obsidian**: 中二病、ゴスロリ、デジタル・ネクロマンサー。
    - **Cursor**: 活発な妹、ツッコミ役、保護者。
    - 2人の掛け合いやプロフィールが見れるセクション。
- **Works / Links**:
    - Noteの漫画記事へのリンクをカード形式で配置。
    - サムネイル画像と共に「読む」ボタンを設置。

## 3. Tech Stack (技術スタック)
- **Core**: HTML5, Vanilla JavaScript
- **Styling**: Vanilla CSS (Modern CSS: Flexbox, Grid, CSS Variables)
- **Build Tool**: Vite
